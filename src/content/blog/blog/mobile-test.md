## 代码块测试

```javascript
// 响应式设计的示例代码
function adjustLayout() {
  const width = window.innerWidth;
  if (width < 768) {
    // 移动端布局
    return 'mobile';
  } else if (width < 1024) {
    // 平板布局
    return 'tablet';
  } else {
    // 桌面布局
    return 'desktop';
  }
}
```
